Last Mile Visibility and Decision Support
Vassar Labs brings together data from Sensors, Satellite Data, Drone, Mobile Applications, SCADA, Web Scraped Data and Mathematical & Hydrological Models to provide Last Mile Visibility and Decision Support .


Seamless Connectivity
Vassar Platform brings data coming from different systems via Cell connectivity, Zigbee, LoRa, Web Services, Web Crawlers and third party systems in a seamless way to ensure the systems are integrated and communicate across platforms.


Big Data Architecture
Vassar platform provides a Horizontally scalable Message Bus and Micro Services based processing that provides for real time processing of large data sets and batch processing of historical or archived data with a horizontally scalable Big data database.

Machine Learning and Custom Mathematical models
Vassar platform provides easy pluggability for Predictive Analytical Models, Custom Algorithms, Hydrology models, Crop Specific models and Supervised and Unsupervised Machine Learning models along with Big Data Analytics.

Cloud/Hybrid Enabled Multi-Tenant Solutions
All Vassar Solutions are built on Cloud on Horizontally scalable resilient distributed systems architecture.

Presentation Layer
An easy to configure presentation layer that works seamlessly between different browsers and devices.
